A dependency of using Thumbprint React or SCSS components is the inclusion of the `thumbprint-global-css` package.

The output of this package is a small CSS file and it should be the first file included in your Thumbprint SCSS or CSS project.

## SCSS usage

```scss
@import '[node_modules path]/@thumbtack/thumbprint-global-css/thumbprint-global';
```

## What's included

-   **reset**: Normalize styling for root all HTML elements.
-   **body**: Color, font, and line-height defaults for the `body` element.
-   **a**: Color and hover states default for the `a` element.
